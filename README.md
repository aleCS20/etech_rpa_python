# Projeto RPA: Automação de Testes com Selenium e Python

## 📝 Descrição
Este projeto faz parte do curso de **RPA e Agentes de IA para a Indústria 4.0** (Fundação Paulo Feitosa). O objetivo é realizar a automação do **Test Case 1: Register User** do site [Automation Exercise](http://automationexercise.com), aplicando conceitos de **POO (Programação Orientada a Objetos)**, **SOLID** e o padrão **Page Object Model (POM)**.

## 🏗️ Estrutura do Projeto
- `src/main.py`: Ponto de entrada do sistema.
- `src/pages/`: Camada de interação com a interface (Page Objects).
- `src/tests/`: Scripts de orquestração dos casos de teste.
- `src/data/`: Massa de dados (Mock) para preenchimento de formulários.
- `src/utils/`: Constantes globais e configurações de driver.

## 🛠️ Tecnologias Utilizadas
- **Linguagem:** Python 3.x
- **Framework:** Selenium WebDriver
- **Gerenciador de Driver:** WebDriver Manager (Chrome)
- **Ambiente:** Linux Mint / Windows
